# claude-essence

Personal Claude Code toolset — output styles, skills, and self-evolving commands for principled engineering.

## Install

Add the marketplace:

```bash
/plugin marketplace add blai/claude-essence
```

Then install:

```bash
/plugin install claude-essence@blai
```

Or test locally:

```bash
claude --plugin-dir /path/to/claude-essence
```

## What's Included

### Output Styles

- **minimalist** — Principal engineer mindset. Zero bloat. Systems thinking. Challenge everything.

### Skills

- **model-hierarchy** — Route tasks to the cheapest Claude model. Target: 80% Haiku / 15% Sonnet / 5% Opus.

### Commands

- **/evolve** — Self-improve any tool in this plugin by researching best practices and applying enhancements.
